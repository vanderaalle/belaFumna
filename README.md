# Bela Fumna

A Bela-based stompbox project — Andrea Valle, 2025/26.

Research and development of an effects pedal built around the [Bela](https://bela.io) platform (BeagleBone Black + cape), running SuperCollider.

## Contents

- `bela_stompbox_tips_state_of_art.md` — tips, tricks, and resources for building a Bela stompbox
- `bela_pedal_bibliography.md` — bibliography of Bela-based effects pedals and related resources
- `bela_fumna_sticker.svg` — project sticker
- `tester.scd` — SuperCollider test patch
